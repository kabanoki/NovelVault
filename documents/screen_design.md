## 3. 各画面の詳細

### S-01 メイン画面

**役割**：アプリのホーム。サイドバーで階層を辿り、メインエリアで本文を読む。

**レイアウト**

```
┌────────────────────────────────────────────────────────────────┐
│ [+ サイト追加] [🔍 検索...]                  [★お気に入り] [⚙設定] │
├──────────────────┬─────────────────────────────────────────────┤
│ サイドバー          │  メインエリア                                  │
│                  │                                              │
│ ▼ サイトA          │   作品タイトル                                  │
│   ▼ 作品1         │   第3話「再会」                ★お気に入り       │
│     第1話         │   ─────────────────────────────              │
│     第2話 ★       │                                              │
│   ▶ 作品2         │   本文がここに表示される...                       │
│ ▶ サイトB          │                                              │
│                  │                                              │
│                  │   [← 前のページ]              [次のページ →]    │
└──────────────────┴─────────────────────────────────────────────┘
```

**主要操作**
- サイドバーでサイト・作品の展開／折りたたみ
- ページクリックで本文表示
- 右上「+ サイト追加」→ S-02 起動
- サイト/作品/ページの行アクション、または右クリックメニューで編集・削除・URL取得
- 本文上部の★アイコンでお気に入り登録/解除
- 検索バー入力で S-06 へ遷移

**Phase 2 実装メモ**
- まず行アクションボタンで編集・削除を実装する
- 右クリックメニューは同じ編集・削除処理を呼び出す後続UIとして追加する

**呼び出すコマンド**
- `siteList` / `workListBySite` / `pageListByWork`（サイドバー構築）
- `pageGet`（本文表示）
- `favoriteCheck` / `favoriteAdd` / `favoriteRemove`（★ボタン）

---

### S-02 サイト登録/編集モーダル

**役割**：サイトの新規登録・既存サイトの編集。

**フォーム項目**

| 項目 | 型 | 必須 | 備考 |
|------|----|----|------|
| サイト名 | text | 必須 | 表示用の名称 |
| base_url | text | 必須 | サイトのベースURL |

**ボタン**：[キャンセル] [保存]

**呼び出すコマンド**：`siteCreate` / `siteUpdate`

---

### S-03 作品登録/編集モーダル

**役割**：作品の新規登録・既存作品の編集。

**フォーム項目**

| 項目 | 型 | 必須 | 備考 |
|------|----|----|------|
| サイト | select | 必須 | 既存サイトから選択 |
| 作品タイトル | text | 必須 | |
| 著者名 | text | 任意 | |
| 説明 | textarea | 任意 | |
| 取得元URL | text | 任意 | 作品トップページURL |
| プロファイル | select | 任意 | サイト内のプロファイルから選択（Phase 6〜） |

**ボタン**：[キャンセル] [保存]

**呼び出すコマンド**：`workCreate` / `workUpdate` / `siteProfileList`

---

### S-04 ページ手動登録/編集モーダル

**役割**：ページの手動登録（URL未取得）・既存ページの編集。

**フォーム項目**

| 項目 | 型 | 必須 | 備考 |
|------|----|----|------|
| 作品 | select | 必須 | 既存作品から選択 |
| 章番号 | number | 任意 | サイト上の章番号 |
| タイトル | text | 任意 | |
| 取得元URL | text | 任意 | |
| URL種別 | radio | 必須 | normal / wayback / local |
| 文字コード | select | 任意 | auto / utf-8 / shift_jis / euc-jp |

**ボタン**：[キャンセル] [保存] [保存して取得（S-05へ）]

**呼び出すコマンド**：`pageCreate` / `pageUpdate`

---

### S-05 URL取得ダイアログ

**役割**：URLからHTMLを取得し、CSSセレクタで本文を抽出してページに保存する。

**フォーム項目**

| 項目 | 型 | 必須 | 備考 |
|------|----|----|------|
| 対象ページ | label | - | 既存pageの情報を表示 |
| URL | text | 必須 | デフォルトはpages.source_url |
| URL種別 | radio | 必須 | normal / wayback / local |
| タイトル用セレクタ | text | 必須 | 例：`h1` |
| 本文用セレクタ | text | 必須 | 例：`div#honbun` |
| 除外セレクタ | text配列 | 任意 | 例：`script`, `.nav` |
| 文字コード | select | 必須 | デフォルト utf-8 |

**ボタン**：[キャンセル] [取得実行]

**取得後の表示**
- 成功：success ステータスとプレビュー、[保存] [再取得] ボタン
- 失敗：fetch_failed / parse_failed のエラー詳細、[再試行] ボタン

**呼び出すコマンド**：`fetchPageByUrl`

### S-11 設定画面（Phase 10）

**役割**：アプリ全体のグローバル設定。

**設定項目（MVP）**
- リーダーフォントサイズ（小／中／大）
- 選択中ページのテキストエクスポート
- SQLite DBバックアップ
- 重複URL診断（既存DBに残る同一作品・同一URLのページを一覧）

**後続検討**
- 既定 User-Agent
- 既定リクエストウェイト（ms）
- データ復元

**呼び出すコマンド**
- `exportPageText`
- `backupDatabase`
- `duplicateSourceUrlList`

## 4. ナビゲーション設計

### サイドバーの状態管理

サイドバーは以下の状態を保持する：

| 状態 | 内容 |
|------|------|
| 展開済みサイトID | Set<number> |
| 展開済み作品ID | Set<number> |
| 選択中ページID | number \| null |

検索結果やお気に入りからのジャンプ時、サイドバーを自動展開して該当ページを強調する。

### キーボードショートカット（最低限）

| キー | 動作 |
|------|-----|
| `⌘ + F` | 検索バーにフォーカス |
| `⌘ + D` | 現在のページをお気に入り登録/解除 |
| `←` / `→` | 前/次ページ（本文表示中） |
| `Esc` | モーダル/ダイアログを閉じる |

## 6. 共通ルール

### モーダル vs フルページ

- **モーダル**：S-02, S-03, S-04, S-05, S-12（短時間で完結する操作）
- **フルページ切替**：S-06, S-07, S-08, S-09, S-10, S-11（じっくり閲覧・編集する操作）

### エラー表示

- フォーム入力エラー：該当フィールド直下に赤字表示
- 取得失敗等の処理エラー：トースト通知 + 詳細はステータスカラムに保存

### 空状態（Empty State）

各リスト画面で空のときは以下を表示：

| 画面 | 空状態メッセージ |
|------|---------------|
| S-01 サイドバー | 「+ サイト追加 からはじめましょう」 |
| S-06/S-07 検索結果 | 「該当する結果がありません」 |
| S-09 お気に入り | 「お気に入りに登録したページがここに表示されます」 |
